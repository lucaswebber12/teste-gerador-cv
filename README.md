#  Gerador de Currículo em PHP (Atividade Prática Orientada)

Este projeto é uma atividade prática da disciplina Fundamentos de Programação para Internet do curso de Análise e Desenvolvimento de Sistemas da UNIPAR EAD. O objetivo é criar um gerador de currículos dinâmico utilizando um stack de desenvolvimento web completo.

---

Tecnologias Utilizadas

O projeto foi construído seguindo os requisitos da atividade, utilizando:

* Ambiente: XAMPP (Apache)
* Backend: PHP (para processar os dados e gerar o currículo)
* Frontend: HTML5 e CSS3
* Framework CSS: Bootstrap 5 (para layout responsivo e componentes de UI)
* JavaScript (Frontend):
 *jQuery: Para manipulação do DOM e adição de campos dinâmicos.
 *JavaScript Puro: Para lógicas como o cálculo automático de idade.
*Versionamento: Git e GitHub

---

Funcionalidades Principais

* Formulário moderno e responsivo (Bootstrap).
* Cálculo automático de idade a partir da data de nascimento (JavaScript).
* Adição dinâmica de campos para:
 * Experiências Profissionais
 * Formação Acadêmica
 * Referências Pessoais
* Geração de um currículo em HTML limpo e pronto para impressão.
* Função de "Salvar como PDF / Imprimir" (`window.print()`).

---

Como Executar o Projeto

1.  Certifique-se de ter o **XAMPP** instalado e o módulo **Apache** em execução.
2. Clone ou baixe este repositório para dentro da pasta `htdocs` do seu XAMPP. (Ex: `C:\xampp\htdocs\gerador-cv`)
3. Abra seu navegador e acesse: `http://localhost/gerador-cv`
4.  Preencha o formulário e clique em "Gerar Currículo".
